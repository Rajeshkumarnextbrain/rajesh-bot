## Instructions

1. Fetch current event data

2. Fetch comparison data:
   - yesterday
   - last week

3. Compare patterns:
   - Sudden spikes
   - Rare events appearing
   - Unusual drops

4. Use get_event_types if needed to understand categories

5. Identify:
   - High-risk anomalies
   - Unexpected behavior

6. Output:
   - What is abnormal
   - Why it matters
